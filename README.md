# CodeAlpha Sudoku Solver

## 📌 Project Description

This project solves a Sudoku puzzle using the Backtracking Algorithm.

## 🚀 Features

- 9×9 Sudoku Grid
- Backtracking Algorithm
- Row Validation
- Column Validation
- 3×3 Grid Validation
- Recursive Solution

## 🛠️ Technologies Used

- C++
- Backtracking
- Recursion

## ▶️ How to Run

Compile:

```bash
g++ sudoku.cpp -o sudoku
```

Run:

```bash
./sudoku
```
